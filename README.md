Raphtaliya-Mark-1 is an open-source transformer-based language model being built completely from scratch using PyTorch.

Unlike projects that depend on existing pretrained models, Raphtaliya is designed to understand and implement every core component of a modern language model from the ground up.

This project focuses on learning, research, experimentation, and building a strong AI foundation for future generations of Raphtaliya.
